# Sistema de Inventario Deportivo

## Descripción

Sistema web para la gestión de inventario de un almacén deportivo desarrollado con React, TypeScript, Spring Boot y MySQL.

## Funcionalidades

* CRUD de productos
* CRUD de usuarios
* Gestión de entradas y salidas
* Reportes gráficos
* Gestión de inventario
* Roles de usuario
* UI/UX responsive

## Tecnologías

### Frontend

* React
* TypeScript
* TailwindCSS
* Vite

### Backend

* Spring Boot
* Java
* MySQL

---

# Guía de instalación en Windows

## Requisitos

* Node.js
* Java JDK 17+
* MySQL
* Git

## Backend

1. Abrir proyecto backend en IntelliJ o VSCode
2. Configurar MySQL en application.properties
3. Ejecutar Spring Boot
4. Verificar:
   http://localhost:8080

## Frontend

1. Abrir terminal
2. Instalar dependencias:

npm install

3. Ejecutar proyecto:

npm run dev

4. Abrir:

http://localhost:3000/
